Hinweise zur Matrix:

Start-Optionen:

-noclock    : Versteckt die Uhr beim Starten
-R<0-100>   : (ohne <>) Stellt den Rotanteil ein
-G<0-100>   : (ohne <>) Stellt den Grünanteil ein
-B<0-100>   : (ohne <>) Stellt den Blauanteil ein

Tasten-Belegungen:
r   : Stellt die Farbe auf rot
g   : Stellt die Farbe auf grün
b   : Stellt die Farbe auf blau
z   : Schaltet den zufälligen Farbwechsel ein/aus
s   : Stellt auf die Anfangsfarbe zurück
c   : Blendet die Uhr ein/aus
esc : Beendet das Programm
